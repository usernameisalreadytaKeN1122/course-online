## course-online：基于Java的在线视频课程系统
在线视频课程系统以提供编程课程、学习IT技能为设计目的，整合多所高校、多个平台的精品课程。首页显性地展示了系统所包含的全部课程，用户可点击主页、全部课程、关于我们和更多按钮显示不同的页面。
